<td>ἀντί + μόνος<br />
(<i>anti monos</i>);<br />
<i>antimonium</i>/<br />
<i>antimonie</i><sup id="cite_ref-3" class="reference"><a href="#cite_note-3"><span>[</span>3<span>]</span></a></sup></td>
<td>various</td>
<td></td>
<td>Possibly from Greek ἀντί + μόνος (<i>anti monos</i>), approximately meaning "opposed to solitude", as believed never to exist in pure form, or ἀντί + μοναχός (<i>anti monachos</i>) for "monk-killer" (in French folk etymology, anti-moine "monk's bane"), because many early alchemists were monks, and antimony is poisonous. This may also be derived from the <a href="/wiki/Pharaonic" title="Pharaonic" class="mw-redirect">Pharaonic</a> (<a href="/wiki/Ancient_Egypt" title="Ancient Egypt">ancient Egyptian</a>), <i>Antos <a href="/wiki/Ammon" title="Ammon">Ammon</a></i> (expression), which could be translated as "bloom of the god Ammo".<br />
The symbol <b>Sb</b> is from Latin name <b>Stibium</b>, which is derived from Greek Στίβι <i>stíbi</i>, a variant of <i>στίμμι</i> <i>stimmi</i> (genitive: στίμμεος or στίμμιδος), probably a loan word from Arabic or Egyptian

<td>33</td>
<td><a href="/wiki/Arsenic" title="Arsenic">Arsenic</a></td>
<td>As</td>
<td><a href="/wiki/Syriac_language" title="Syriac language">Syriac</a>/<a href="/wiki/Persian_language" title="Persian language">Persian</a> via Greek, Latin, <a href="/wiki/Old_French" title="Old French">Old French</a>, and <a href="/wiki/Middle_English" title="Middle English">Middle English</a></td>
<td>ἀρσενικόν (<i>arsenikon</i>)</td>
<td>orpiment</td>
<td>Greek <a href="/wiki/Male" title="Male"><i><b>a</b>r<b>s</b>enikon</i></a></td>
<td>From Greek ἀρσενικόν (<i>arsenikon</i>), which is adapted from the <a href="/wiki/Syriac_language" title="Syriac language">Syriac</a> ܠܐ ܙܐܦܢܝܐ (al) zarniqa<sup id="cite_ref-etymonline.com_5-0" class="reference"><a href="#cite_note-etymonline.com-5"><span>[</span>5<span>]</span></a></sup> and <a href="/wiki/Persian_language" title="Persian language">Persian</a>, زرنيخ (<i>zarnik</i>), "yellow <a href="/wiki/Orpiment" title="Orpiment">orpiment</a>". Ἀρσενικόν (<i>arsenikon</i>) is <a href="/wiki/False_etymology" title="False etymology">paretymologically</a> related to the Greek word ἀρσενικός (<i>arsenikos</i>), which means "masculine" or "potent." These words were adapted as the Latin <i>arsenicum</i> and Old French <i>arsenic</i>, which is the source for the English arsenic.<sup id="cite_ref-etymonline.com_5-1" class="reference"><a href="#cite_note-etymonline.com-5"><span>[</span>5<span>]</span></a></sup></td>

<td>56</td>
<td><a href="/wiki/Barium" title="Barium">Barium</a></td>
<td>Ba</td>
<td>Greek via <a href="/wiki/Modern_Latin" title="Modern Latin" class="mw-redirect">Modern Latin</a></td>
<td>βαρύς (<i>barys</i>)</td>
<td>heavy</td>
<td>Greek <a href="/wiki/Density" title="Density"><i><b>ba</b>rys</i></a></td>
<td>βαρύς (<i>barys</i>) means "heavy". The oxide was initially called "barote", then "baryta", which was modified to "barium" to describe the metal. <a href="/wiki/Humphry_Davy" title="Humphry Davy">Sir Humphry Davy</a> gave the element this name because it was originally found in <a href="/wiki/Baryte" title="Baryte">baryte</a>, which shares the same source.<sup id="cite_ref-7" class="reference"><a href="#cite_note-7"><span>[</span>7<span>]</span></a></sup></td>
</tr>
<tr>
<td>97</td>
<td><a href="/wiki/Berkelium" title="Berkelium">Berkelium</a></td>
<td>Bk</td>
<td><a href="/wiki/Anglo-Saxon" title="Anglo-Saxon" class="mw-redirect">Anglo-Saxon</a> via English</td>
<td><a href="/wiki/University_of_California,_Berkeley" title="University of California, Berkeley">University of California, Berkeley</a></td>
<td></td>
<td>toponym: <a href="/wiki/Berkeley,_California" title="Berkeley, California"><b>B</b>er<b>k</b>eley, California</a></td>
<td>Named for the <a href="/wiki/University_of_California,_Berkeley" title="University of California, Berkeley">University of California, Berkeley</a>, where it was discovered. <a href="/wiki/Berkeley,_California" title="Berkeley, California">Berkeley, California</a>, was named in honor of <a href="/wiki/George_Berkeley" title="George Berkeley">George Berkeley</a>. "Berkeley" is derived from <a href="/wiki/Old_English" title="Old English">Old English</a> <i>beorce léah</i>, which meaning <i>birch lea.</i><sup id="cite_ref-8" class="reference"><a href="#cite_note-8"><span>[</span>8<span>]</span></a></sup></td>
</tr>
<tr>
<td>4</td>
<td><a href="/wiki/Beryllium" title="Beryllium">Beryllium</a></td>
<td>Be</td>
<td><a href="/wiki/Sanskrit" title="Sanskrit">Sanskrit</a>, <a href="/wiki/Pali" title="Pali">Pali</a>, and <a href="/wiki/Prakrit" title="Prakrit">Prakrit</a> via Greek, Latin, <a href="/wiki/Old_French" title="Old French">Old French</a>, and <a href="/wiki/Middle_English" title="Middle English">Middle English</a></td>
<td>βήρυλλος (<i>beryllos</i>)</td>
<td>a blue-green spar, but possibly related to the name of <a href="/wiki/Belur" title="Belur">Belur</a></td>
<td>descriptive (colour): <a href="/wiki/Beryl" title="Beryl"><b>be</b>ryl</a></td>
<td>βήρυλλος <i>beryllos</i>, denoting <a href="/wiki/Beryl" title="Beryl">beryl</a>, which contains beryllium.<sup id="cite_ref-9" class="reference"><a href="#cite_note-9"><span>[</span>9<span>]</span></a></sup> The word is derived (via Latin: Beryllus and French: Béryl) from the Greek <i>βήρυλλος</i>, <i>bērullos</i>, <i>a blue-green spar</i>, from <a href="/wiki/Prakrit" title="Prakrit">Prakrit</a> <i>veruliya</i> (वॆरुलिय‌), from <a href="/wiki/Pali" title="Pali">Pāli</a> <i>veḷuriya</i> (वेलुरिय); <i>veḷiru</i> (भेलिरु) or, <i>viḷar</i> (भिलर्), "to become pale," in reference to the pale semiprecious gemstone <a href="/wiki/Beryl" title="Beryl">beryl</a>.<sup id="cite_ref-10" class="reference"><a href="#cite_note-10"><span>[</span>10<span>]</span></a></sup> The word is ultimately derived from the <a href="/wiki/Sanskrit" title="Sanskrit">Sanskrit</a> word वैडूर्य vaidurya- which might be related to the name of the city of <a href="/wiki/Belur" title="Belur">Belur</a>.<sup id="cite_ref-11" class="reference"><a href="#cite_note-11"><span>[</span>11<span>]</span></a></sup></td>
</tr>
<tr>
<td>83</td>
<td><a href="/wiki/Bismuth" title="Bismuth">Bismuth</a></td>
<td>Bi</td>
<td><a href="/wiki/New_Latin" title="New Latin">Modern Latin</a> from German</td>
<td><i>bisemutum</i></td>
<td>white mass</td>
<td>descriptive (colour): <b>bi</b>semutum</td>
<td><i>bisemutum</i> is derived from German <i>Wismuth</i>, perhaps from <i>weiße Masse</i>, and means "white mass", due to its appearance.</td>

<td></td>

<td></td>
<td>eponym: <a href="/wiki/Niels_Bohr" title="Niels Bohr">Niels <b>B</b>o<b>h</b>r</a></td>
<td>Named in honor of <a href="/wiki/Niels_Bohr" title="Niels Bohr">Niels Bohr</a>, who made fundamental the understanding of <a href="/wiki/Atomic_structure" title="Atomic structure" class="mw-redirect">atomic structure</a> and <a href="/wiki/Quantum_mechanics" title="Quantum mechanics">quantum mechanics</a>.<sup id="cite_ref-naming_controversy_12-0" class="reference"><a href="#cite_note-naming_controversy-12"><span>[</span>12<span>]</span></a></sup></td>
</tr>
<tr>
<td>5</td>
<td><a href="/wiki/Boron" title="Boron">Boron</a></td>
<td>B</td>
<td><a href="/wiki/Persian_language" title="Persian language">Persian</a> via <a href="/wiki/Arabic_language" title="Arabic language">Arabic</a>, <a href="/wiki/Medieval_Latin" title="Medieval Latin">Medieval Latin</a>, <a href="/wiki/Anglo-Norman" title="Anglo-Norman">Anglo-Norman</a>, <a href="/wiki/Middle_French" title="Middle French">Middle French</a>, and <a href="/wiki/Middle_English" title="Middle English">Middle English</a></td>
<td>بورق (<i>buraq</i>)</td>
<td></td>
<td>Latin <a href="/wiki/Borax" title="Borax"><b>b</b>orax</a> from Arabic</td>
<td>بورق (<i>buraq</i>) is derived from the <a href="/wiki/Persian_language" title="Persian language">Persian</a>, بوره (<i>burah</i>), which refers to <a href="/wiki/Borax" title="Borax">borax</a>. These terms were adapted as <a href="/wiki/Medieval_Latin" title="Medieval Latin">Medieval Latin</a> <i>baurach,</i> <a href="/wiki/Anglo-Norman" title="Anglo-Norman">Anglo-Norman</a> <i>boreis,</i> and <a href="/wiki/Middle_English" title="Middle English">Middle English</a> <i>boras,</i> which became the source of the English boron.</td>
</tr>
<tr>
<td>35</td>
<td><a href="/wiki/Bromine" title="Bromine">Bromine</a></td>
<td>Br</td>
